# How to make your own test?

## Step 1: Upload Image
Upload scanned image using the (+) button in ```home``` tab
Give it a name

## Step 2: Create Test
Long click the image and ```Create Test```

## Step 3: Wait for the test to be created in the ```Dashboard``` tab

## Step 4: Thats All!!
